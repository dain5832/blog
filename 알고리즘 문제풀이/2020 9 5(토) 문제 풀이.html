문제 출처:

프로그래머스 코딩테스트 연습 > 해시 > Level 1 완주하지 못한 선수

https://programmers.co.kr/learn/courses/30/lessons/42576

문제 요약:

마라톤 참가선수 명단과 마라톤을 완주한 선수 명단을 비교하여, 완주하지 못한 선수의 이름을 출력. 단, 동명이인이 있을 수 있고, 완주하지 못한 선수는 항상 1명이다.

접근 방법:

처음에는 set을 사용하여 차집합을 하고자 했는데, 그렇게 하면 동명이인은 경우에 틀린 답이 나오기 때문에, 그냥 두 리스트의 원소를 하나씩 비교해야겠다는 생각을 하였다.

처음에는 .remove를 사용했으나 효율성 점수가 깎여 통과하지 못하였고, 검색한 결과 complexity가 O(n)이라는 것을 알게되어 pop을 사용하는 것으로 바꾸었다.

최종적으로 두 리스트를 sort한 뒤 마지막 원소를 비교하고, 비교한 원소를 pop으로 제거하는 방식을 사용하였다.

풀이 코드:

def solution(participant, completion):
    participant.sort()
    completion.sort()
    while completion:
        if participant[-1] == completion[-1]:
            participant.pop()
            completion.pop()
        else:
            answer = participant[-1]
            return answer
    return participant[0]

다른 사람의 풀이:

def solution(participant, completion):
    answer = ''
    temp = 0
    dic = {}
    for part in participant:
        dic[hash(part)] = part
        temp += int(hash(part))
    for com in completion:
        temp -= hash(com)
    answer = dic[temp]

    return answer
def solution(participant, completion):
    participant.sort()
    completion.sort()
    for i in range(len(completion)):
        if participant[i] != completion[i]:
            return participant[i]
    return participant[len(participant)-1]

배운 점:

1) dictionary와 hash를 활용한 풀이.

→ 두 리스트의 일치여부를 비교할 때, 각 원소를 hash한 뒤 그 총합이 같은가를 통해 비교한다!

2) 구지 마지막 원소를 비교하지 않고 그냥 순차적으로 비교해도 됐을텐데 하는 생각이 든다. 그렇게 하면 if 조건을 일치하지 않을 때로 할 수 있으니 코드를 더 간결하게 짤 수 있었을 것.
